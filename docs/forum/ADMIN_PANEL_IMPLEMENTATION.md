# Admin Panel Forum - Plan Implementare

## Structură Generală

Admin Panel-ul va avea următoarele secțiuni principale (tabs/sidebar):

1. **Dashboard** - Statistici live, grafice activitate
2. **Moderare** - Ban, mute, delete, shadow ban, istoric restricții
3. **Reputație** - Admin Award, vizualizare toate log-urile
4. **Badge-uri** - Acordare badge-uri manuale
5. **Rapoarte Braconaj** - Aprobare/respingere, status tracking
6. **Roluri** - Gestionare roluri utilizatori
7. **Marketplace** - Verificare vânzători
