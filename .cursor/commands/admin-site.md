# admin-site

Generează și actualizează doar Admin Panel-ul site-ului principal (FishTrophy).
Acest panel este complet separat de forum și gestionează întregul ecosistem al site-ului: conturi, permisiuni, capturi, recorduri, echipamente, bannere, parteneriate și configurări globale.

Admin Site trebuie să permită editarea COMPLETĂ a profilului oricărui utilizator:
- avatar, bannere și imagini din profil
- descriere, bio, interese
- echipamente din profil
- recorduri personale și capturi încărcate
- date personale (email, nume, telefon) doar dacă este necesar
- ștergerea conținutului ofensator sau nepotrivit
Toate editările trebuie logate (cine, ce, când).

Funcții suplimentare:
- gestionare utilizatori și permisiuni globale
- verificare firme / conturi comerciale
- gestionare recorduri oficiale și capturi
- gestionare echipamente din platformă
- gestionare proiecte comunitare (ex: populări, curățări)
- gestionare conținut, articole și bannere
- configurări site globale

UI-ul trebuie să fie profesionist, organizat pe secțiuni mari, clar structurat și ușor de folosit.
Nu amesteca acest admin panel cu funcțiile adminului de forum.
